# 日本京都赏樱攻略

京都的樱花季通常在3月下旬至4月上旬。

## 1. 最佳赏樱点
- **哲学之道**: 沿着运河的小径，两旁种满了樱花树。
- **清水寺**: 著名的古寺，夜樱非常壮观。
- **岚山**: 渡月桥周边的樱花与山景相映成趣。
- **圆山公园**: 京都市内最古老的公园，有一棵巨大的“祇园枝垂樱”。

## 2. 住宿建议
- 樱花季住宿非常紧张，建议提前半年预订。
- 推荐体验传统的日式旅馆（Ryokan）。

## 3. 礼仪
- 赏樱时请勿攀折花枝。
- 垃圾请自行带走。
