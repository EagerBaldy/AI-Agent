# 巴黎旅行指南：浪漫之都

巴黎，法国的首都，也是世界艺术与时尚的中心。

## 1. 必去景点
- **埃菲尔铁塔 (Eiffel Tower)**: 巴黎的地标，建议提前网上购票登顶。
- **卢浮宫 (Louvre Museum)**: 世界四大博物馆之首，镇馆之宝《蒙娜丽莎》。
- **凯旋门 (Arc de Triomphe)**: 位于香榭丽舍大街西端。
- **圣母院 (Notre Dame)**: 哥特式建筑的代表（修复中）。

## 2. 美食推荐
- **法式长棍 (Baguette)**: 随处可见的面包店。
- **马卡龙 (Macaron)**: Ladurée 是最著名的品牌。
- **鹅肝 (Foie Gras)**: 经典法餐前菜。

## 3. 交通贴士
- 地铁（Metro）是最方便的交通工具，覆盖全城。
- 注意保管财物，防范扒手。
