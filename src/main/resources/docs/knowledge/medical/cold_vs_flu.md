# 常见感冒与流感的区别

普通感冒（Cold）和流行性感冒（Flu）症状相似，但流感通常更严重。

## 1. 症状对比
| 症状 | 普通感冒 | 流感 |
| :--- | :--- | :--- |
| **发热** | 少见，低热 | 常见，高热 (39-40℃) |
| **头痛** | 少见 | 常见且剧烈 |
| **全身酸痛** | 轻微 | 常见且严重 |
| **疲乏虚弱** | 轻微 | 早期出现，持续时间长 |
| **鼻塞/流涕** | 常见 | 有时出现 |
| **咳嗽** | 轻度至中度 | 常见，可能严重 |

## 2. 治疗建议
- **感冒**: 多休息，多喝水，对症治疗（退烧药、止咳药）。通常7-10天自愈。
- **流感**: 尽早使用抗病毒药物（如奥司他韦），特别是高危人群。

## 3. 预防
- 接种流感疫苗是预防流感最有效的手段。
- 勤洗手，保持通风。
