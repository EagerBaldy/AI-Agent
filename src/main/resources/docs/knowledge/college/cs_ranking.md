# 计算机科学 (CS) 专业选校指南

CS 是目前最热门的专业之一，选校时需综合考虑排名、地理位置和课程设置。

## 1. 美国顶尖院校 (Tier 1)
- **MIT (麻省理工)**: 理论与工程并重，AI 领域极强。
- **Stanford (斯坦福)**: 硅谷核心，创业氛围浓厚。
- **CMU (卡内基梅隆)**: CS 独立成院，方向最全，SCS 是金字招牌。
- **UC Berkeley (伯克利)**: 公立第一，系统方向强。

## 2. 选校维度
### 2.1 地理位置
- **硅谷 (Bay Area)**: 就业机会最多，科技公司云集。
- **西雅图 (Seattle)**: 微软、亚马逊总部。
- **波士顿/纽约**: 金融科技、生物科技机会多。

### 2.2 课程偏好
- 偏科研：选择 PhD 项目多、实验室强的学校。
- 偏就业：选择 Co-op 项目丰富、校友网络广的学校（如 Waterloo, Northeastern）。

## 3. 申请建议
- 高 GPA 和 GRE 是基础。
- 相关的科研经历或实习经历是加分项。
- 强有力的推荐信。
