我将为您优化前端界面，实现基于助手类型的主题定制和首页滚动功能。

### 核心实施计划

#### 1. 首页滚动优化
* **修改 `App.vue` 样式**：为 `.home-container` 增加 `overflow-y: auto` 属性，并调整布局方式（移除强制居中），确保内容超出屏幕时可以自然滚动，同时不影响工作台的固定布局。

#### 2. 主题定制系统 (基于 CSS 变量)
* **定义主题配置**：
    * **编程助手**：科技蓝 (`#007bff`) + 机器人风格头像。
    * **旅游助手**：自然绿 (`#00b894`) + 地球风格头像。
    * **作文名师**：活力橙 (`#e17055`) + 笔/书本风格头像。
    * **素材百科**：优雅紫 (`#6c5ce7`) + 书籍风格头像。

* **注入 CSS 变量**：
    * 在 `App.vue` 中根据 `currentMode` 动态计算 CSS 变量（如 `--primary-color`, `--avatar-bg`），并绑定到根元素。

* **组件样式适配**：
    * **`App.vue`**：侧边栏选中态、Logo、标题文字颜色跟随主题。
    * **`ChatMessage.vue`**：AI 头像背景、用户消息气泡背景跟随主题。
    * **`ChatInput.vue`**：发送按钮背景、输入框聚焦边框跟随主题。

### 预期效果
* **首页**：内容较多时可流畅上下滚动。
* **编程助手**：保持现有的科技蓝风格。
* **旅游助手**：切换为清新的绿色主题，营造轻松氛围。
* **作文助手**：切换为温暖的橙色主题，模拟批改作业的笔触颜色。
* **素材助手**：切换为沉稳的紫色主题，体现知识的厚重感。
