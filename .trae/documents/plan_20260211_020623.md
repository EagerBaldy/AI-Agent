要实现“AI 旅游规划小助手”，我们需要在现有项目中扩展一个新的 AI 服务，专门用于处理旅游相关的咨询和规划。

### 核心思路
1.  **定义新的 AI 服务接口 (`AiTravelHelperService`)**：类似于现有的 `AiCodeHelperService`，但专注于旅游领域。
2.  **创建专属的角色设定 (System Prompt)**：为旅游助手编写专门的提示词，让它扮演专业的旅游规划师。
3.  **配置 AI 服务工厂 (`AiCodeHelperServiceFactory`)**：注册并构建新的旅游 AI 服务实例。
4.  **新增 API 接口 (`AiController`)**：暴露新的 HTTP 接口供前端调用。

### 具体实施步骤

#### 1. 创建 System Prompt 提示词文件
在 `src/main/resources` 下新建 `system-prompt-travel.txt`，定义旅游助手的角色设定：
```text
你是一位经验丰富、热情友好的专业旅游规划师。
你的职责是根据用户的需求（如目的地、预算、时间、兴趣偏好等），提供个性化的旅游建议、行程规划和实用贴士。
请以结构清晰、语气轻松的方式回答，可以包含景点推荐、美食介绍、交通指南和住宿建议。
如果用户没有提供足够的信息，请礼貌地询问更多细节以便提供更好的服务。
```

#### 2. 定义 AI 服务接口
新建接口 `com.yupi.aicodehelper.ai.AiTravelHelperService`：
```java
public interface AiTravelHelperService {
    // 基础对话
    @SystemMessage(fromResource = "system-prompt-travel.txt")
    String chat(String userMessage);

    // 流式对话（支持打字机效果）
    @SystemMessage(fromResource = "system-prompt-travel.txt")
    Flux<String> chatStream(@MemoryId int memoryId, @UserMessage String userMessage);
}
```

#### 3. 注册 Bean 到工厂类
修改 `com.yupi.aicodehelper.ai.AiCodeHelperServiceFactory`，添加构建 `AiTravelHelperService` 的逻辑：
```java
@Bean
public AiTravelHelperService aiTravelHelperService() {
    return AiServices.builder(AiTravelHelperService.class)
            .chatModel(myQwenChatModel)
            .streamingChatModel(qwenStreamingChatModel)
            .chatMemoryProvider(memoryId -> MessageWindowChatMemory.withMaxMessages(10))
            // 暂时不绑定代码相关的工具（如面试题工具），保持纯净的旅游咨询体验
            .build();
}
```

#### 4. 暴露 API 接口
修改 `com.yupi.aicodehelper.controller.AiController`，注入新服务并添加接口：
```java
@Resource
private AiTravelHelperService aiTravelHelperService;

@GetMapping("/travel/chat")
public Flux<ServerSentEvent<String>> travelChat(int memoryId, String message) {
    return aiTravelHelperService.chatStream(memoryId, message)
            .map(chunk -> ServerSentEvent.<String>builder()
                    .data(chunk)
                    .build());
}
```

### 验证计划
1.  启动后端服务。
2.  使用浏览器或 Postman 访问 `http://localhost:8081/api/ai/travel/chat?memoryId=1&message=我想去云南玩5天，有什么推荐？`。
3.  确认返回的内容是旅游规划相关的建议，且以流式方式输出。
